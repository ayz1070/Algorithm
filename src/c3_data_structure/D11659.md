# 1. 문제 분석하기
- 1억번의 연산 허용, n은 최대 100000 -> O(nlogn)
- 예) 1,3 인 경우 5 + 4 + 3 = 12
- 예) 2,4 인 경우 4 + 3 + 2 = 9
- 일반화
- i, i+n 번째 -> arr[i-1] + arr[i] + ... + arr[i+n-1] 
- O(n*m) 의 시간복잡도이므로 실패 -> 구간합 활용
- 입력 받을 때 sum 배열 
- 예) sum -> 5, 9, 12, 14, 15
- 1, 3인 경우, 12-0
- 2, 4인 경우, 14-5
- 일반화
    - result = sum[i] - sum[j-1]
    - sum[0] = 0

# 2. 수도 코드
~~~
n, m 입력
sums[n+1] 배열 초기화
sums[0] = 0

for(i=1부터 n+1까지){
    num = 정수 입력
    sums[i] = sum[i-1]+num;
}
for(m 만큼){
    i,j 입력
    result = sum[j-1] - sum[i]
    print(result)
}
